# Live-timer-
dit is een project die ik thuis heb geprogrammeerd. Het is een applicatie die een live timer laat zien van de dag, een project waar ik mee kan oefenen om timers te implemteren in projecten.
3 timers met elke een eigen kleur voor de teller, het is van links naar rechts met het aantal uur, het aantal minuten en het aantal seconden.
Mijn motivatie hiervoor was om mijn skill te oefenen, zodat ik het in mijn eigen projecten kan toevoegen
